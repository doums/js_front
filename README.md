## COOLCHAT69

###### :cloud: a realtime web chat :cloud:

#### main features
 - Built with [webpack](https://webpack.js.org/)
 - Hot Module Replacement (HMR)
 - React Hot Loader
 - ESLint
 - DevServer (webpack-dev-server)
 - Source Maps

#### download
```
git clone https://github.com/doums/js_front.git
```

#### installation
```
npm i
```

#### run for development
```
npm start
```

#### build for production
```
npm run build
```
