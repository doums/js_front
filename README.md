## COOLCHAT69

###### :cloud: a realtime web chat, client :cloud:

#### prerequisites
- git
- node (npm)

#### download
```
git clone https://github.com/doums/js_front.git
```

#### installation
```
npm i
```

#### run for development
```
npm start
```

#### build for production
```
npm run build
```

#### lint
```
npm run lint
```
